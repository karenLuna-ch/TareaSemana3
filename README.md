  # Sistema de Gestión de Mascotas

**Nombre:** Karen  Chango

## Descripción

Este proyecto contiene dos programas desarrollados en Python.

### Programación Tradicional

Se desarrolló un programa utilizando funciones, variables y entrada de datos por teclado para registrar y mostrar la información de una mascota.

### Programación Orientada a Objetos

Se desarrolló una clase llamada Mascota con los atributos nombre, especie y edad. Además, se implementaron los métodos mostrar_informacion() y hacer_sonido(), creando dos objetos para demostrar su funcionamiento.

## Reflexión

La Programación Tradicional utiliza funciones y variables para resolver problemas. La Programación Orientada a Objetos permite organizar mejor la información mediante clases y objetos, facilitando la reutilización del código y haciendo que los programas sean más fáciles de mantener y entender.
